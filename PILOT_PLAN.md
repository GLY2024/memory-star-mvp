# 记忆星河试点方案
## 高校离退休中心合作计划

---

## 一、试点目标

验证AI辅助回忆录撰写产品在目标用户群体中的接受度和产品-市场匹配度（PMF）。

**核心指标：**
- 完成率：≥70%用户完成至少5次访谈
- 满意度：NPS ≥ 50
- 转介绍率：≥30%用户愿意推荐给朋友

---

## 二、合作模式

### 方案A：公益合作（推荐）

| 项目 | 内容 |
|------|------|
| **我方提供** | 免费回忆录撰写服务（限10-20位老人） |
| **中心提供** | 场地、招募、现场协助 |
| **成果共享** | 口述历史资料（脱敏后可用于研究） |
| **周期** | 2-3个月 |

**优势：**
- 降低中心决策门槛
- 积累真实用户案例
- 建立信任关系

### 方案B：服务采购

| 项目 | 内容 |
|------|------|
| **服务内容** | 回忆录撰写服务包 |
| **价格** | ¥1,999/人（机构价，原价¥4,999） |
| **数量** | 10人起订 |
| **增值服务** | 集体回忆录、历史讲座 |

## 三、执行计划

### Phase 1: 筹备期（2周）

**Week 1-2: 签约准备**
- [ ] 拜访中心负责人，介绍项目
- [ ] 确定合作模式和试点规模
- [ ] 签署合作协议
- [ ] 准备宣传物料

**交付物：**
- 合作协议
- 宣传海报/推文
- 报名登记表

### Phase 2: 招募期（2周）

**Week 3-4: 用户招募**
- [ ] 中心发布招募信息
- [ ] 线下宣讲会（可选）
- [ ] 收集报名信息
- [ ] 筛选首批用户（10-20人）

**筛选标准：**
- 表达能力较好
- 有意愿记录人生故事
- 能配合完成5次以上访谈
- 子女支持（重要）

### Phase 3: 服务期（6-8周）

**Week 5-12: 访谈服务**

| 周次 | 内容 | 形式 |
|------|------|------|
| 5 | 开场+基本信息收集 | 现场，专员协助 |
| 6-7 | 童年+求学经历 | 线上/现场 |
| 8-9 | 工作+婚姻家庭 | 线上/现场 |
| 10-11 | 退休生活+人生感悟 | 线上/现场 |
| 12 | 审阅定稿 | 现场 |

**人员配置：**
- 推广专员：1人（中心提供或我方派驻）
- AI访谈：自动化
- 人工审核：1人（我方）
- 现场支持：中心工作人员

### Phase 4: 总结期（2周）

**Week 13-14: 成果交付**
- [ ] 回忆录印刷交付
- [ ] 用户满意度调研
- [ ] 案例整理
- [ ] 总结会议

**交付物：**
- 精装回忆录（每人1本）
- 试点总结报告
- 用户证言视频（如有）

## 四、推广专员工作手册

### 职责
1. 协助老人使用APP/设备
2. 解答操作疑问
3. 收集反馈
4. 维护关系

### 话术示例

**开场：**
> "张教授您好！我是记忆星河的小李。我们是一个帮助老同志记录人生故事的项目。您看您什么时候方便，我帮您试试这个语音聊天功能？"

**引导：**
> "您就对着手机说话，像跟我聊天一样。AI助手会问您一些问题，您慢慢回答就行。"

**鼓励：**
> "您刚才讲的那段特别好！关于那个年代的细节，以后您的孙子孙女看到一定会很感动的。"

## 五、风险与应对

| 风险 | 可能性 | 应对策略 |
|------|--------|----------|
| 老人不会用APP | 高 | 专员现场协助，简化流程 |
| 老人讲述不连贯 | 中 | AI引导提问，多次访谈补充 |
| 中途退出 | 中 | 设置里程碑奖励，情感关怀 |
| 隐私顾虑 | 中 | 明确数据使用范围，签署协议 |
| 质量问题 | 低 | 人工审核兜底 |
